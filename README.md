# ordekBorc - Simple Lending/Borrowing App

A simple web application for managing money lending and borrowing between friends.

## Features
- Lend money to friends
- Borrow money from friends
- Accept/Reject/Counter-offer options for lending/borrowing requests
- Track your transactions

## Setup Instructions

1. Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

4. Open your browser and go to `http://localhost:5000`

## Project Structure
- `app.py`: Main application file
- `models/`: Database models
- `templates/`: HTML templates
- `static/`: CSS and JavaScript files
